🌍 Earthquake Application
AI-Powered Disaster Risk Analysis & Simulation System
📌 Overview

Earthquake Application is an AI-powered desktop system designed to monitor earthquakes in Turkey and worldwide, analyze regional risk levels, and simulate potential disaster scenarios.

The project focuses on risk awareness, decision support, and simulation-based analysis through an interactive and multi-language interface.

🚀 Core Features

🌐 Multi-language interface (TR / EN / ES / AR / FR)

🧠 AI-based earthquake risk scoring system

🗺️ Interactive global earthquake map

📊 City-based risk visualization table

🔄 Scenario simulation mode

📡 Live earthquake data integration

🧠 AI-Based Risk Analysis

The system evaluates earthquake risk levels using machine learning models trained on historical and environmental datasets.

Outputs:

-Risk Score (0–1 scale)

-Risk Category (Low / Medium / High)

-Model architecture and training pipeline are kept private.
